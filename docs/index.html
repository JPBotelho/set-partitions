Set partitions

In this website are resources on counting set partitions and related structures.

It is not a complete reference. These are just some of the things I learned about when studying this topic. I made this website because it would've been of great help to me.

There might be mistakes, double check with the given sources when in doubt.

If there is something you feel should be added, removed or changed, contact me at jpbotelho.costa (at) gmail.com

Named restrictions

Bell numbers

  • Bell Number - with no restrictions

  • Associated Bell number - ≥m\geq m≥m elements per block

  • Restricted Bell number - ≤m\leq m≤m elements per block

  • r-Bell number - the first rrr elements are in separate blocks

  • Restricted r-Bell number - the first rrr elements are in separate blocks, ≤m\leq m≤m elements per block

  • Associated r-Bell number - the first rrr elements are in separate blocks, ≥m\geq m≥m elements per block

  • Reduced Bell number - elements with pairwise distance <d<d<d are in separate blocks

Stirling numbers

All of these are assumed to be Stirling numbers of the second king, ommitted to keep the titles short.

  • Stirling number of the second kind - kkk blocks

  • Associated Stirling number - kkk blocks and ≥m\geq m≥m elements per block

  • Restricted Stirling number - kkk blocks and ≤m\leq m≤m elements per block

  • r-Stirling number - kkk blocks, the first rrr elements are in separate blocks

  • Associated r-Stirling number - kkk blocks, the first rrr elements are in separate blocks, ≥m\geq m≥m elements per block

  • Restricted r-Stirling number - kkk blocks, the first rrr elements are in separate blocks, ≤m\leq m≤m elements per block

  • Reduced Stirling number - kkk blocks,elements with pairwise distance <d< d<d are in separate blocks

Other restrictions

  • ≥m\geq m≥m elements per block, and the largest block has jjj elements

  • Into blocks of equal size

  • Into blocks of distinct sizes

Restriction on specific elements

  • Non-crossing partitions

  • Non-nesting partitions

  • Such that the blocks are ordered but the elements in them aren't -> ordered Bell / Fubini numbers

  • Into non-crossing partitions -> Catalan number

  • With block lengths given by the $k$th integer partition of the number of elements -> A036040

  • Such that the smallest block has size kkk -> A182930

  • Even number of blocks, odd block size -> A003709
