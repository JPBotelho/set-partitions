Reduced Stirling numbers of the second kind

Sd(n,k)S^d(n, k) Sd(n,k)

is the number of ways of partitioning the set {1,2,...,n}\{1, 2, ..., n\}{1,2,...,n} into kkk subsets such that in each subset, elements have pairwise distance at least ddd.
